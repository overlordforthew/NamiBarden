「春の夜に聴く 心の大掃除瞑想」
〜 眠りながら、心と体の荷物をおろす 〜

ナミ・バーデン

========================================
【BGM指示】穏やかなピアノ or アンビエント。春の夜のイメージ。
虫の声、そよ風の音をうっすらミックスしてもOK。
全体を通してゆっくり、間をたっぷり取って読む。
「...」= 3〜5秒の間  「（間）」= 5〜8秒の間  「（長い間）」= 8〜12秒の間
========================================


──────────────────────────────
【第一章】導入 〜 春の夜へようこそ（約8分）
──────────────────────────────

こんばんは。

今日も一日、お疲れさまでした。

...

今夜は、あなたのために
特別な時間を用意しました。

春の夜の、静かな瞑想の時間です。

...

まず、楽な姿勢を見つけてください。

仰向けでも、横向きでも、
あなたが一番リラックスできる姿勢で大丈夫です。

...

目を閉じて...

そして、ゆっくりと
深い呼吸をしてみましょう。

（間）

鼻からゆっくり吸って...

...

口からふーっと吐いて...

（間）

もう一度。

鼻から吸って...
お腹がふわっと膨らむのを感じて...

...

口から、長く、ゆっくり吐いて...
体の力が抜けていくのを感じます。

（間）

もう一回だけ。

深く吸って...

...

全部、吐き切って...

（長い間）

はい、もう頑張らなくて大丈夫です。

呼吸は自然なリズムに戻してください。
コントロールしなくていい。
体が勝手に呼吸してくれます。

...

今夜のテーマは「心の大掃除」。

春は、日本では昔から
新しい始まりの季節ですね。

桜が咲いて、
新しい風が吹いて、
自然が生まれ変わる季節。

...

でも、新しいものを迎え入れるためには
まず、スペースを作る必要があります。

お部屋の大掃除と同じように、
心にも、体にも、
いつの間にか溜まったものがあります。

...

今夜は、そのいらなくなったものを
ひとつずつ、やさしく手放していきましょう。

無理に捨てるのではありません。

「もう持たなくていいよ」と
自分に許可を出す時間です。

（長い間）

では、まず想像してみてください。

あなたは今、
春の夜の、美しい場所にいます。

...

満開の桜の木の下。

大きくて、立派な桜の木です。

見上げると、
枝いっぱいに淡いピンクの花が咲いていて、
その向こうに、丸い月が静かに光っています。

...

春の夜風が、やさしく頬を撫でていきます。

冷たくもなく、暑くもなく、
ちょうどいい温かさ。

...

桜の花びらが、ひらひらと
舞い降りてきます。

一枚、また一枚。

あなたの周りに、静かに降り積もっていく
薄桃色の花びら。

...

ここは、完全に安全な場所です。
誰にも邪魔されません。
何も求められません。

ただ、この桜の木の下で
春の夜を感じていてください。

（長い間）


──────────────────────────────
【第二章】体の大掃除 〜 冬の重さを溶かす（約12分）
──────────────────────────────

それでは、体の大掃除を始めましょう。

冬の間、私たちの体には
いろいろなものが溜まっています。

寒さから体を守るために
ぎゅっと縮こまっていた筋肉。

忙しさの中で無意識に力が入っていた場所。

疲れ。冷え。こわばり。

...

今から、春の温かい風が
あなたの体を通り抜けていきます。

その風が触れた場所から、
冬の間に溜まったものが
すーっと溶けていきます。

...

まず、足の先に意識を向けてください。

つま先。足の裏。かかと。

...

春の温かい風が、足の先を包みます。

冷えていた足先が、じんわりと温かくなっていく。

冬の間、あなたの体を支えてくれた足。
たくさん歩いて、たくさん踏ん張ってくれた足。

「ありがとう。もう力を抜いていいよ」

...

足の疲れが、溶けて流れていきます。

（間）

温かい風が、ふくらはぎへ...
膝へ... 太ももへ...

ゆっくり、ゆっくり上がっていきます。

...

脚全体が、ぽかぽかと温かくなって、
重力に身を任せるように
ベッドに沈んでいきます。

力が抜けて...
もう何も支えなくていい。

（間）

次に、腰に温かい風が届きます。

腰は、いつも頑張っている場所ですね。

座っている時も、立っている時も、
あなたの体を支え続けてくれています。

...

「お疲れさま。今は休んでいいよ」

腰の奥にあった緊張が、
ふわっとほどけていきます。

じんわり、じんわり。

（間）

温かさが、お腹に広がります。

お腹は、感情を溜め込みやすい場所です。

不安を感じた時、緊張した時、
お腹がキュッとなったこと、ありませんか？

...

今、そのお腹の力を、全部抜いてください。

ふーっと。

お腹が柔らかくなって、
呼吸がさらに深くなっていくのを感じます。

溜まっていた緊張が、息と一緒に出ていく。

（長い間）

春の風が、胸に届きます。

胸。心臓のあたり。

...

ここには、いろいろな感情が住んでいますね。

嬉しかったこと。悲しかったこと。
誰かを想う気持ち。
言えなかった言葉。

...

今は、何も整理しなくていいです。

ただ、胸のあたりが
温かい光に包まれているのを感じてください。

ぽかぽかと、優しい温もり。

その温もりの中で、
かたくなっていた胸が
少しずつ、ほぐれていきます。

（長い間）

温かさが、両方の肩に流れていきます。

肩。

...

多くの人が、一番力を入れている場所。

責任。プレッシャー。期待。
全部、肩に乗せてきましたね。

...

今、その肩を
ストンと落としてください。

...

もっと落として。

...

まだ少し力が入っているかもしれません。

もう一段階、落として。

...

はい。そこです。

肩の荷物が、するすると滑り落ちていく。
もう持たなくていい。

（間）

温かさが腕を通って、手の先まで届きます。

指先がじんわり温かい。

手のひらが、ぽかぽかしている。

（間）

そして、首から顔へ。

首の後ろの緊張がほどけて...

あごの力が抜けて...

歯の噛み締めをゆるめて...

...

頬の筋肉がゆるんで...

眉間のしわが消えて...

おでこが広く、平らになって...

...

頭のてっぺんまで、
春の温かい風が通り抜けていきます。

（長い間）

今、あなたの体全体が
春の温かさに包まれています。

冬の間に溜まっていた重さ、冷え、こわばり。

全部、溶けて流れていきました。

体が軽い。

ベッドに溶けていくような感覚。

ふわふわと、雲の上に浮かんでいるような。

...

体の大掃除が終わりました。

すっきりと、軽くなった体を感じてください。

（長い間）


──────────────────────────────
【第三章】心の大掃除 〜 春風に乗せて手放す（約15分）
──────────────────────────────

体が軽くなったところで、
次は、心の大掃除をしていきましょう。

...

桜の木の下に、あなたは静かに横たわっています。

花びらが降り続けている。

月の光が、やさしく照らしている。

...

あなたの目の前に、
小さな春風が生まれました。

この風は、特別な風です。

あなたが手放したいものを
乗せて、遠くへ運んでくれる風。

...

では、この一年を
少し振り返ってみましょう。

無理に思い出さなくていいです。

ただ、心の中を静かに覗いてみてください。

何が浮かんできますか？

（長い間）

もしかしたら、
人間関係の疲れがあるかもしれません。

誰かに気を遣いすぎて、疲れたこと。

分かってもらえなくて、悲しかったこと。

傷つく言葉を言われて、
まだ胸のどこかに残っていること。

...

もしそういうものがあったら、
それを一枚の桜の花びらに乗せてください。

ふわっと、花びらの上に置くだけでいい。

...

そして、春風がそれを
やさしく運んでいきます。

ひらひらと、ひらひらと、
月の光の中を遠くへ...

...

手放すのは、忘れることじゃないんです。

「もう、これに縛られなくていい」
と、自分に許可を出すこと。

...

その出来事は、あなたの人生の一部です。
消す必要はありません。

でも、重荷として
持ち続ける必要もないんです。

（長い間）

次に。

この一年で、
我慢してきたことはありますか？

...

本当は嫌だったけど、
笑顔で引き受けたこと。

本当は休みたかったけど、
頑張り続けたこと。

本当は「助けて」と言いたかったけど、
一人で乗り越えようとしたこと。

...

あなたは、とても強い人ですね。

でも、強い人ほど、
たくさんのものを溜め込んでしまう。

...

その我慢を、
もう一枚の花びらに乗せてください。

「私は十分頑張った。
 もう、これは持たなくていい。」

...

春風が、やさしく運んでいきます。

花びらが、月明かりの中を
ゆっくり、ゆっくり遠ざかっていく。

（長い間）

それから。

自分を責めてしまったこと、ありませんか？

...

「もっとこうすればよかった」

「あの時、ああ言えばよかった」

「なんで私はこうなんだろう」

...

そういう声が、
頭の中で何度も繰り返されることがあります。

でも、ちょっと想像してみてください。

もし、あなたの大切な人が
同じことで自分を責めていたら、
あなたは何と言いますか？

...

きっと、
「そんなに自分を責めないで」
「あなたは十分頑張ってるよ」
「大丈夫だよ」
って、言いますよね。

...

今夜は、その言葉を
自分自身にかけてあげてください。

...

「私は、十分頑張っている。」

「あの時の私は、あれが精一杯だった。」

「それで、いいんだよ。」

...

自分を責める声を、
花びらに乗せて、手放しましょう。

春風が、受け取ってくれます。

（長い間）

最後にもうひとつ。

将来への不安。

「これからどうなるんだろう」
「このままでいいのかな」
「もう遅いんじゃないか」

...

春は、始まりの季節。

でも、新しいことが始まる時、
不安を感じるのは自然なことです。

桜だって、冬の寒さの中で
じっと耐えて、じっと準備をして、
そして春になって、やっと花を咲かせます。

あなたの中にも、
まだ咲いていない花があります。

それは、ちゃんと準備をしている。
時が来たら、必ず咲きます。

...

だから、焦らなくていい。

不安も、花びらに乗せて、
春風に渡してしまいましょう。

...

ひらひら、ひらひら...

月の光の向こうへ...

消えていきます。

（長い間）

あなたの心から、
たくさんのものが旅立っていきました。

花びらに乗って、春風に乗って、
月の光の中へ。

...

心が、すーっと軽くなっているのを
感じてください。

まるで、部屋の窓を全部開けて
新しい風を通したような、
あの気持ちよさ。

すっきりと、さっぱりと、
心に新しいスペースができました。

（長い間）


──────────────────────────────
【第四章】新しい空間 〜 春のエネルギーを迎え入れる（約10分）
──────────────────────────────

心が軽くなって、
新しいスペースができました。

...

桜の花びらが降り続けています。

月の光が、あなたを照らしています。

春の夜風が、やさしく吹いています。

...

今、空になったスペースに
新しいものが入ってこようとしています。

春のエネルギー。

新しい季節の、
みずみずしい、生命力に溢れたエネルギー。

...

胸のあたりに、
小さな温かい光が生まれるのを感じてください。

最初は、ろうそくの炎くらい。
小さくて、でも確かに温かい。

...

その光が、少しずつ大きくなっていきます。

オレンジ色の、やさしい光。

胸から、お腹へ。
肩へ。腕へ。

ゆっくり、ゆっくり広がって...

...

体全体が、温かい光に満たされていきます。

春の光。新しいエネルギー。

あなたの中に、
新しい季節が始まっています。

（長い間）

その光の中で、
ひとつだけ、自分に聞いてみてください。

...

「この春、私はどんな自分でいたい？」

...

答えは、すぐに出なくていいです。

言葉にならなくてもいい。

ただ、なんとなくの感覚...
ふわっとしたイメージ...
それで十分です。

（長い間）

もしかしたら、
もっと穏やかな自分かもしれない。

もっと自由な自分かもしれない。

もっと正直な自分かもしれない。

もっと優しい自分かもしれない。

...

どんな答えでも、大丈夫です。

それが、あなたの春の種です。

今、胸の温かい光の中に
その種が植えられました。

...

この種は、焦らなくていい。

毎日少しずつ、水をあげるように、
自分を大切にしていれば、
いつか芽が出て、花が咲きます。

...

桜も、一日で咲くわけじゃない。

長い冬を越えて、
少しずつ、少しずつ準備をして、
ある日ふわっと咲くんです。

あなたも、同じ。

大丈夫。

ちゃんと、咲きます。

（長い間）

桜の木の下で、
あなたは今、とても穏やかな気持ちでいます。

体は軽く、心はすっきりして、
胸には温かい光が灯っている。

...

もう何も考えなくていい。
何も頑張らなくていい。

ただ、この穏やかさの中にいてください。

（長い間）


──────────────────────────────
【第五章】眠りへ 〜 桜の花びらの毛布（約10分〜フェードアウト）
──────────────────────────────

さて、そろそろ眠りの時間です。

...

桜の花びらが、
あなたの上にふわりふわりと降り積もって、
やわらかな毛布のようになっています。

花びらの毛布。

ほんのり桜の香りがして、
とても温かくて、軽い。

...

この毛布に包まれて、
あなたは安心して眠ることができます。

...

月の光が、あなたを見守っています。

桜の木が、あなたを守っています。

春の夜風が、子守唄のように
そっと吹いています。

...

体がどんどん重くなっていきます。

でも、心は羽のように軽い。

...

呼吸は、もう自然のまま。

吸って...吐いて...

吸って...吐いて...

（間）

少しずつ、意識がぼんやりしてきます。

それでいいんです。

そのまま、眠りに身を委ねてください。

...

今日、手放したもの。

体の疲れ...
人間関係の重さ...
我慢してきたもの...
自分を責める声...
将来への不安...

全部、春風が連れて行ってくれました。

...

そして、新しいスペースには
温かい光と、春の種。

明日の朝、目が覚めた時、
あなたは少し軽くなっているはずです。

...

今は、もう何も考えなくていい。

ただ、桜の花びらの毛布に包まれて...

月の光に見守られて...

春の夜に、溶けていくように...

...

おやすみなさい。

...

今日も一日、
よく頑張りましたね。

...

おやすみなさい...

...

（長い間）

（声がゆっくりフェードアウト）

...
